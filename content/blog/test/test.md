# 1. Repository 생성하기

![github](/images/github.png)

GitHub Blog를 만들려면 Github에 Repository를 생성해야 합니다.

github-blog.png

GitHub에 로그인 한 뒤에 우측 상단에 있는 New Repository 버튼을 클릭하면 repository 생성 페이지로 이동하게 됩니다. 이 때 Import a repository 버튼을 클릭합니다.

github-blog-1.png
